# MVP V1 (최소 실행 기능)

## 목표
MVP V1은 하나의 완전한 콘텐츠 루프에 집중합니다: 노트 생성 → 공개/비공개 제어 → 검색 → 북마크.

## 백로그

| 우선순위 | 기능 | 포함 범위 | 제외 범위 (나중에) | 완료 조건 |
|---|---|---|---|---|
| P0 | 노트 CRUD | 제목, 본문, 태그, 공개/비공개 | 파일 첨부, 버전 비교 | 생성/조회/수정/삭제가 API로 동작, 테스트 통과 |
| P0 | 공개/비공개 제어 | 노트 단위 `PUBLIC`/`PRIVATE` | 팀 공유 권한 | 비공개 노트는 인증 없이 접근 불가 |
| P0 | 검색 | 제목/본문/태그 키워드 + 페이징 | 고급 랭킹, 맞춤법 교정 | 키워드로 올바른 결과 반환, 페이징 동작 |
| P0 | 북마크 | 북마크 토글 + 목록 조회 | 폴더 계층, 우선순위 정렬 | 멱등성 보장, 중복 방지 |
| P1 | 요약 | 요약문 저장/조회 (규칙 기반) | LLM 자동요약 고도화 | 요약 생성/갱신 가능, 길이 검증 |
| P1 | 기본 보안 | 로그인 세션 또는 토큰 기반 | OAuth 소셜 로그인, RBAC | 비공개 API는 인증 필수 |

## MVP 완료 기준
- 한 번의 사용 흐름(생성 → 조회 → 검색 → 북마크)을 데모로 보여줄 수 있다.
- 비공개 콘텐츠는 인증 없이 접근 불가능하다.
- 핵심 API가 테스트로 검증되고 문서화되었다.
